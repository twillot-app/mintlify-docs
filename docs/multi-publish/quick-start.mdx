完成一键发布内容，只需要以下几个步骤：

1. 安装扩展
2. 登录谷歌帐号
3. 创建或者选择草稿
4. 连接发布平台
5. 选择发布平台
6. 发布

## 安装

前往谷歌 Chrome 浏览器应用商店下载安装 [Twillot Multi Publish](https://s.twillot.com/multi-publish) 扩展，安装以后您无需进行任何配置。
安装完成以后扩展会自动打开一个页面，在这里您就可以开始您的发布之旅了。
后续也可以点击浏览器右上角的扩展图标来打开这个页面。

## 登录

使用 Twillot Multi Publish 发布内容需要您先登录您的 Google 账号，登录以后您就可以开始发布内容了。

### 创建或者选择草稿

打开 [Twillot Multi Publish](https://www.twillot.com/multi-publish/drafts?utm_source=docs&isNewDraftOpen=true) 页面，你可以开始输入你的文章标题和内容，输入完成后点击“Publish Across Platforms”按钮前往平台选择页面。

> 注意：如果您之前已经创建过草稿，那么您可以在草稿列表中选择您想要发布的草稿。

### 连接发布平台

在打开的 [Twillot Multi Publish](https://www.twillot.com/multi-publish/platforms?utm_source=docs) 页面中，默认会展示所有支持的平台，初次进入需要手动连接平台。只需点击右侧的连接 icon，等待几秒即可。

> 连接成功，连接状态 icon 会变成绿色，表示连接成功。红色则表示连接失败。

### 选择发布平台

在平台选择页面，您可以选择您想要发布的平台，选择完成后点击“Start Publishing”按钮开始发布。

> 注意：如果您的平台没有连接成功，是无法进行选择的。

### 发布

发布后，你将进入 Twiilot Multi Publish 任务列表页面，在这里你可以看到你当前正在进行的发布任务，也可以看到你已经完成的任务。

> 注意：如果发布失败，您可以点击任务列表中的 “Retry” Icon 重新发布。
