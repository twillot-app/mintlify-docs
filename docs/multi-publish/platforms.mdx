## 选择平台的原则

Twillot Multi Publish 的核心目的是帮助 Indie Makers 快速发布内容建立反向链接，因此我们选择平台的原则是：

1. 平台允许自由发布文本/图片等内容，包括链接
2. 平台 Domain Rating 高，流量大，平台活跃
3. 平台支持发布包含反向链接的内容

## 支持的平台

Twillot Multi Publish 目前支持以下平台：

- GitHub
- Medium
- Dev.to
- Kofi
- Patreon

这四个平台 Domain Rating 都是 90 以上，都是目前非常流行的平台。
这四个平台允许自由发布文本/图片等内容，包括链接，因此非常适合 Indie Makers 发布内容建立反向链接。

> 切记您发布的内容也需要遵守平台规则。
