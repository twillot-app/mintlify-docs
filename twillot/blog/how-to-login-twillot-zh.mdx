## Twillot Bookmarks for X 是什么?

Twillot Bookmarks for X 是一款 Chrome 扩展工具，专为 X（原 Twitter）用户设计，帮助他们轻松管理和导出推特书签。通过 Twillot，用户可以快速搜索、分类、以及导出他们的推特书签，从而更好地管理自己的推特内容。

## 为什么你需要 Twillot Bookmarks for X?

如果你是一个活跃的 X 用户，可能会保存大量的推特书签，以备将来参考。然而，X 本身并没有提供便捷的工具来管理这些书签。最基本的搜索功能也是七月份才姗姗来迟。Twillot Bookmarks for X 正是为了解决这个问题，它可以帮助你：

- 快速搜索特定的书签
- 将书签按类别整理
- 一键导出所有书签以备份或在其他平台上使用

## 如何安装 Twillot Bookmarks for X?

### 直接下载

https://s.twillot.com/x-bookmarks

### 商店搜索下载

1. 打开 Chrome 浏览器并访问 Chrome 网上应用店。
2. 在搜索栏中输入 “Twillot Bookmarks for X” 并按下 Enter。
3. 在搜索结果中找到并点击 “Twillot Bookmarks for X”。
4. 点击 “添加到 Chrome” 按钮并确认安装。

由于 Edge 浏览器的审核进度非常缓慢，推荐 Edge 浏览器直接从 Chrome 商店安装扩展。
其它浏览器暂时不支持，如果您有需要，请在 [社区](https://x.com/i/communities/1796857620672008306) 中提出。

安装完成后，你会在浏览器的工具栏中看到一个新的 Twillot 图标。

## 如何登录 Twillot Bookmarks for X?

严格来说，Twillot 并不需要登录，只需要在您登录以后进行授权即可。

1. 确保您已经登录 X 账号。
2. 点击浏览器工具栏中的 Twillot 图标。
3. 在打开的扩展管理页面中，点击 “Authenticate”。
4. 此时会弹出推特书签页面，如果授权成功，您将可以访问所有的书签管理功能。

## 如何使用 Twillot Bookmarks for X 导出 Twitter 书签?

1. 点击浏览器工具栏中的 Twillot 图标。
2. 在打开的扩展管理页面左侧导航中，点击 “Export”。
3. 在打开的导出页面中，选择您想要导出的书签范围，或导出所有书签。
4. 点击 “Export” 按钮，书签将被保存为 CSV 或 JSON 格式的文件。
5. 下载完成后，你可以使用 Excel 或其他工具查看和管理这些书签。

如果您在导出时有自定义的需求，可以考虑购买我们的 Basic 和 Pro 会员。
详情请见 [会员](https://s.twillot.com/x-bookmarks)。
