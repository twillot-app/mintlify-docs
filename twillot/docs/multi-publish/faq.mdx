## Twillot Multi Publish 是什么？

Twillot Multi Publish 是一款营销工具，帮助 Indie Makers 快速发布内容建立反向链接的工具。
它的核心目标是建立反向链接，这有助于提升网站的搜索引擎排名，吸引更多用户访问您的网站。

## Twillot Multi Publish 不是什么？

Twillot Multi Publish 不是一款博客发布工具，如果您专注于打造样式精美的博客文章，请使用其他多平台博客发布工具。

> 当然，您也可以使用 Twillot Multi Publish 发布博客文章，但是您需要自己编写 Markdown 内容，并且不支持富媒体文件发布。代码支持方面也要依赖具体平台的规则。

## 支持发布哪些形式的内容？

您发布的内容需要以 Markdown 形式编写，支持文本、链接等。
Twillot Multi Publish 目前暂不支持图片/视频/文档等富媒体文件的发布，如果您的内容包含富媒体文件，请以链接的形式发布。

> 注意：您发布的内容需要遵守各个发布平台的规则。

## 支持发布哪些平台？

Twillot Multi Publish 目前支持以下平台：

- GitHub
- Medium
- Dev.to
- Kofi
- Patreon

我们计划支持的平台达到几百个，后续会陆续支持更多平台，欢迎关注。
如果您有好的平台推荐，欢迎联系 [twillot@outlook.com](mailto:twillot@outlook.com)。

## 什么是连接发布平台？

连接发布平台仅仅是一个状态监测，用于通知 Twillot Multi Publish 您已经成功连接了发布平台，可以进行后续的发布操作。

## 如何链接发布平台？

在平台列表，点击右侧的连接 Icon，会打开一个目标平台的发布页面，需要您在该平台注册帐号登录，登录成功后连接成功。

> 由于各个平台的登录机制不同，Twillot Multi Publish 可能无法检测您的可发布状态。此时您只需要在登录成功后重新点击连接 Icon 即可。

## 连接发布平台会泄露我的隐私数据吗？

不会，连接发布平台仅仅是监测您是否成功登录了目标平台，不会搜集和上报您的隐私数据。

## 为什么我连接不上发布平台？

确认您是否已经在该平台登录，如果已经登录，请重新点击连接 Icon 即可。

## 为什么我连接上发布平台以后，也没有发布成功

有可能您的登录态已经过期，请重新登录。然后重新点击连接 Icon 即可。

## 为什么不支持富媒体和文件发布？

文件发布涉及到针对每个平台定制开发，过程较为繁琐。后续随着平台规则变动，也需要频繁的维护工作。

Twillot Multi Publish 目前暂不支持图片/视频/音频/文档等富媒体文件的发布，如果您的内容包含富媒体文件，请以链接的形式发布。
